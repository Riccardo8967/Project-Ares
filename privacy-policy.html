Welche Daten werden verarbeitet?

Diese Anwendung verbindet sich mit der WHOOP API und greift auf folgende Daten zu:

  • Trainingsdaten (Aktivitätsart, Dauer, Herzfrequenz)
  • Erholungsdaten (Recovery Score, Herzfrequenzvariabilität, Ruheherzfrequenz)
  • Schlafdaten (Dauer, Start- und Endzeit)
  • Körpermasse (Grösse, Gewicht, maximale Herzfrequenz)
  • Profildaten (Name, Benutzername)

Datenspeicherung

Alle Daten werden ausschliesslich lokal auf deinem Gerät gespeichert (Browser localStorage). Es werden keine Daten an externe Server von Apex Enduro übermittelt.

Datenweitergabe

Deine Daten werden nicht an Dritte weitergegeben.

Deine Rechte

Du kannst deine Daten jederzeit über die App löschen: Einstellungen → Daten → Zurücksetzen.
